# Suggested Commands

- Install dependencies: `npm install`.
- Start local dev server: `npm start` (CRA default `http://localhost:3000`, may choose another port if occupied).
- Production build: `npm run build`.
- Test runner: `npm test` (watch mode by default; set CRA env flags if noninteractive behavior is needed).
- Project AGENTS asks to prefix shell commands with `rtk` when available; in this environment `rtk` was not installed, so raw commands were necessary.
